# Dia 4 - Frontend React

Frontend React + Vite + TypeScript para simular el portal de pagos.

## Requisitos

- Node.js 18+
- Backend activo en http://localhost:3000

## Instalar

```bash
npm install
```

## Ejecutar

```bash
npm run dev
```

## Verificación

- Abrir: http://localhost:5173
- Crear una transferencia y observar estado PENDIENTE -> APROBADO.
